搭建一个个人网站，前期的定位特别重要，你必须要了解你做这个网站的目的，
是为了个人展示、知识分享还是为了兴趣爱好，这直接决定了你将要做的网站的类型和内容风格，
也就是你网站的主题，选定内容方向后，我们再来尝试搭建网站，会起到事半功倍的效果。

免费搭建

如何不购买服务器，域名 来搭建自己的个人网站呢

利用 github pages 服务可以免费搭建一个静态服务。
由于无需购买服务器，所以没有数据库访问，很适合建立自己的博客或者个人网站，而不适合大型网站。

注册GitHub账号，并创建一个代码仓库（repository），起名为GitHub的账号名+".github.io"
例如：我的github名叫xiuyuantech，创建的仓库名 xiuyuantech.github.io
注意一个账号只能创建一个

编写网站代码或者使用开源的网站项目，例如 Hexo, Yilia, Next,也可参考我的这篇文章 hexo-theme-beantech

推荐

对于海外运营，一定要谈谈App Store/Google Play开发者收款方式。

大多数的应用出海个人/公司，Apple store以及 google play在结算的时候，都是打美金到开发者收款账户。美金转换成人民币，必然会有缴纳手续费、汇率转化这些繁琐的流程
如何才能减少成本，简化流程，向你们推荐一个低成本的跨境收款方式- Pingpone。

优点：
1.国内新秀收款机构，无结汇限额，能完美解决你的每年5万额度限制
2.提现工作日当天到账
3.入账免费，提现手续费最高1%
4.使用pingpong成功收款后，还能免费享受vcc（虚拟信用卡）服务，全球线上消费皆可使用
