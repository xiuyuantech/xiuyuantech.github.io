VPN简介 虚拟专用网（Virtual Private Network，VPN）是一种“基于公共数据网，给用户一种直接连接到私人局域网感觉的服务”。
VPN极大地降低了用户的费用，而且提供了比传统方法更强的安全性和可靠性。
其核心功能包括：

  1. ‌加密通信‌：采用IPSec、SSL等协议对传输数据进行加密，防止窃听或篡改。‌‌
  2. ‌远程访问‌：允许用户通过互联网安全连接到企业内网或其他私有网络，如员工出差时访问公司内部系统。‌‌
  3. ‌隐私保护‌：隐藏用户真实IP地址，避免网络活动被追踪。‌‌

为什么需要VPN

  • 数据加密传输，隐藏原始IP地址，避免网络行为被追踪

  • 防监控与审查，绕过流量监控审查

  • 突破网络限制，访问限制内容

  • 远程安全办公，访问内部系统

如何搭建VPN

1、自建服务

  • 购买VPS
    自行购买云服务器，推荐 Vultr VPS。

  • 一键安装 V2Ray
    安装 V2Ray（VMess 协议）: bash <(curl -s -L https://sh.ipchk.io/v2ray.sh)。

  • 安装V2Ray客户端，设置好配置即可

2、直接购买第三方VPN服务

  • 奈思云VPN, 套餐最低10元/月。

  • 疾风云VPN, 套餐最低14元/月。

使用注意

1、‌法律合规性‌

  • 境内使用需遵守《网络安全法》（禁止绕过防火墙访问非法内容）

  • 避免用于破解版权内容等非法用途

2、‌选择建议‌‌

  • 优先选择无日志记录策略的服务商

  • 警惕免费VPN（可能出售用户数据）

Clash Verge 教程

注意事项

Clash Verge 最初的库也跑路了，现在Clash Verge指的是 clash-verge-rev

使用前，请彻底关闭或者卸载你电脑的杀毒，比如 360、火绒、腾讯安全管家，这些垃圾东西会收集你的浏览信息，并且会影响使用。

根据可靠消息，360 会导致用户被逮捕，所以尤其要注意。

在某次更新后，Clash Verge的界面有较大改动，如果你现在还在使用旧版本，建议更新到最新版本。

简介

Clash Verge 是代理工具 Clash 内核的 GUI 图形客户端，支持 Windows、Linux、 MacOS 系统。

自从各 Clash 删库之后，目前 Clash Verge 是少数还可以直接从 GitHub 下载的 Clash 客户端之一，同时支持 Windows、Linux、macOS 三大操作系统，内置集成中文版，直接可以上手使用。

使用教程

第一步：下载 Clash Verge 并安装

如果你可以访问 GitHub，可以直接从 官方仓库 下载最新版本。

如果你无法访问 GitHub，可以从下方的 下载 中下载。

第二步（方案 A）：一键导入订阅

打开 丛雨云主页，点击一键导入 Clash.

如果你没有购买套餐，或者套餐已经过期，订阅链接将不会显示。

浏览器会弹出这样的弹窗，点击不是取消的那个按钮。不同浏览器显示的按钮可能不一样，但必定是点不是取消的那个按钮

现在，打开 Clash Verge 的配置界面，你应该能看到订阅已经导入。

如果没有正常导入，可以尝试手动导入。

第二步（方案 B）：手动导入订阅

打开 丛雨云主页，点击 Clash 家族处的复制按钮。

打开 Clash Verge，点击左侧的「配置」，在上方输入框中粘贴订阅链接，点击「导入」。

第三步（对于较早的clash verge rev）：选中订阅

导入后应能看到如下图所示界面。

点击刚刚导入的链接对应的小卡片，选中订阅链接。如果订阅链接小卡片左侧高亮（如下图），则表明已经选中。

如果你发现在导入之后，订阅链接就已经选中，则跳过这个步骤。在较早版本，clash verge rev 需要手动选中订阅，而在较新的版本，已经解决了这个设计缺陷。

第四步：选择想要的节点

切换到「代理」界面，你会看到这样的

类似于 Clash For Windows，上面有 4 个选项，对应了 Clash 3 种工作模式。

  1. 全局：所有流量都会通过代理服务器，如果你愿意牺牲国内网站速度来换取匿名，可以选择这个。
  2. 规则：根据规则文件以及自选规则，进行分流。推荐使用。
  3. 直连：等同于没开梯子。所有流量都会直接连接到目标服务器。

如果你正在使用旧版 Clash Verge，还会看到有「脚本」这个模式，但是新版已经移除了，因为这个模式没用。

选择你需要的代理模式，然后，在下方选择节点（支持自动选择最快节点）。

常见故障排除：为什么几乎所有的节点都超时了？

任何机场，即使是非常垃圾的机场，所有节点同时故障的概率几乎为0。通常来说，几乎所有节点同时超时是因为系统时间不准确。请检查你的系统时间是否准确。

另外，如果你的网络本来就不通，那当然所有的节点都会超时。

第五步：开启代理

切换到「设置」，切换系统代理的开关为开启状态。

有些时候，系统代理会失灵，此时可以尝试关闭系统代理并打开 TUN 模式。

即使机场的一些节点有足够高的速度很稳定性，但请不要使用机场打需要团队合作的在线游戏，请购买专门的游戏加速器。

更新订阅

在订阅页面中，点击订阅卡片的右上角的刷新按钮即可。
